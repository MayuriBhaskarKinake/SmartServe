SmartServe

About the Project

SmartServe is a web-based platform developed to connect customers with local service providers. The main objective of this project is to make it easier for users to find and book services such as plumbing, electrical work, cleaning, painting, and other household services through a single platform.

The system provides separate modules for Customers, Service Providers, and Administrators. Customers can browse available services and make bookings, while providers can manage their services and view booking requests. The Admin module helps in monitoring users and bookings within the system.

This project was developed using Python Flask for backend development, PostgreSQL for database management, and HTML/CSS for designing the user interface.

Features

- Customer Registration and Login
- Service Provider Registration
- Customer Dashboard
- Provider Dashboard
- Admin Dashboard
- Service Listing and Search
- Booking Management
- User Profile Management
- PostgreSQL Database Integration

Technologies Used

- Python
- Flask
- HTML
- CSS
- PostgreSQL
- pgAdmin

Project Structure

SmartServe follows a simple and organized structure with separate folders for templates, static files, database configuration, and application logic.

Conclusion

SmartServe is a simple and user-friendly platform that helps customers connect with local service providers efficiently. The project demonstrates the use of Flask, PostgreSQL, and web development concepts to build a practical service-booking application.
